# Mine Sweeper game made with vue3 and typescript

## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm run dev
```

### Compiles and minifies for production
```
pnpm run build
```

Live Preview https://mine-sweeper-rho.vercel.app/
